# Async loading item web app  

* Nodejs API backend 
* Vuejs 3 frontend  

## Start  
Deploy:  
`$ docker-compose up --build -d`  

### Local development  
Server: `$ nodemon server.js`  
App: `$ npm run dev` 

## Backend API endpoints 

* `/grid`
* `/grid/item/info_list?item_id`
* `/grid/item/status?item_id`
